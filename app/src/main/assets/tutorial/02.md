## 核心功能进阶

### 多样化截屏模式

SolveX 适配了多种系统环境，你可以在「权限设置」中自由切换：

- **系统录屏**：标准的 Android 录屏接口，兼容性最佳。
- **无障碍截图**：利用无障碍权限获取屏幕内容，无需每次手动确认录屏弹窗。
- **Shizuku (ADB)**：最高效、最稳定的截屏方式，适合极客用户使用。
- **屏幕取字**：完全不产生截图文件。点击悬浮球后手动圈选文字范围，保护隐私的同时降低流量消耗。

### 隐私与安全

我们深知屏幕数据的敏感性，SolveX 提供了一系列隐私保护方案：

- **隐匿模式 (需 Shizuku)**：开启后，SolveX 会在截屏时自动避开系统隐私窗口，并在「最近任务」中隐藏自己。
- **防截屏保护**：开启后，当你在进行系统截屏或屏幕共享时，SolveX 的界面（如结果抽屉）会自动黑屏处理。

### 结果抽屉交互

解析结果会以半屏抽屉的形式优雅展现：

- **沉浸式阅读**：支持 Markdown 渲染，公式和代码块清晰易读。
- **自动滚动**：开启此功能后，AI 流式输出时抽屉会自动滚动到最新位置。
- **快速复制**：长按解析结果或点击底部的复制按钮，一键带走核心答案。
- **历史回溯**：点击抽屉右上角图标，随时翻阅之前的解析记录。
