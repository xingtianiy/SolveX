## 常见问题

### 悬浮球截图时遮挡题目

1. 进入 APP【设置 → 通用配置】页面

2. 找到【隐藏悬浮球】开关并开启

3. 触发截图指令前 200ms 自动隐藏悬浮球，截图完成后自动恢复显示

### 解析结果内公式显示源码或乱码问题

1. 结果页依靠 MathJax 前端内核渲染 LaTeX 公式，内核依赖网络加载字体与渲染脚本

2. 切换稳定 WiFi / 手机流量，退出详情页重新进入刷新内核

### AI 模型识别题目答案偏差大

1. 含几何图形、函数图像：则需要切换至【多模态视觉引擎】

2. 文本引擎无法读取图片像素图形，仅能进行纯文字识别

### 获取屏幕截图失败，权限异常

1. 检查系统录屏权限是否授予，临时授权会后台自动回收

2. 系统权限冲突（小米 / 鸿蒙定制系统）

3. 关闭后台屏幕录制、投屏、录屏类第三方软件

4. 系统接口失效，切换兜底抓取模式

- 无障碍抓取模式（无需录屏权限，兼容性最强）
- Shizuku 授权抓取模式（适合 Android 12+ 封闭系统）

### 后台一段时间后悬浮球自动消失

1. 安卓定制系统后台杀进程导致，进入手机系统设置

2. 开启 SolveX【允许后台活动】【电池无限制】【自启动】三项权限，即可解决该问题

### 检查更新提示网络异常

1. 检查设备网络连接是否正常，尝试切换 WiFi / 移动数据

2. 部分网络环境可能屏蔽 GitHub，系统会自动切换至 Gitee 或 JsDelivr 源

### 悬浮球大小如何调整

1. 进入 APP【设置 → 通用配置】页面

2. 在「显示设置」分组找到「悬浮球大小」滑块

3. 拖动滑块即可实时调整悬浮球尺寸（24dp ~ 64dp）

### 多图模式切换后悬浮球答案没变化

1. 多图逐张模式下，悬浮球显示的答案与当前查看的图片页对应

2. 使用抽屉界面左右箭头翻页后，悬浮球会自动切换显示对应题目的答案

3. 合并模式下悬浮球按 3.5 秒间隔循环滚动显示所有题目的答案

### 多图模式终止后重新截图内容未更新

1. 双击悬浮球终止多图任务后，系统会清除缓冲区和实时解析内容

2. 重新截图即可开始全新分析，不会残留上次的题目数据

### 多图模式通知设置

1. 进入【设置 → 多图模式配置 → 通知与显示】

2. 开启「是否启用通知」开关即可在多图分析完成后收到系统通知

3. 关闭开关则不会发送任何通知

### 多图模式自动弹出抽屉

1. 进入【设置 → 多图模式配置 → 通知与显示】

2. 开启「自动打开抽屉」开关后，开始执行多图分析时将自动弹出实时解析抽屉

### 模型配置后无法识别或报错

1. 检查 API 地址是否正确，结尾不需要携带 `/v1`（系统自动拼接）

2. 检查 API 密钥是否包含多余空格，系统会自动去除首尾空格

3. 进入提供商编辑页面点击「获取模型列表」验证连接是否正常

### 各模式模型配置独立吗

1. 是的，常规学习模式、自动速查模式、多图模式均有独立的模型配置页面

2. 你可以在【设置】中分别进入各模式的配置页面，为不同场景指定不同的模型和超时参数

### 悬浮球自动隐藏如何关闭

1. 进入 APP【设置 → 通用配置】页面

2. 在「显示设置」分组找到「隐藏悬浮球」开关

3. 关闭后悬浮球将始终保持完整显示，不会自动收缩
