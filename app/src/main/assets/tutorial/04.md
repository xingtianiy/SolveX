## 常见问题解答 (FAQ)

在使用 SolveX 的过程中，你可能会遇到一些疑问。这里汇总了最高频的问题及解决方法。

### 1. API 报错排查建议

**Q: 点击解析后提示“API 密钥无效”或“网络连接超时”？**
- **检查地址**：请确认提供商设置中的 API 地址是否包含了多余的 `/v1`。大多数中转服务只需填写到 `.com` 或 `.cn`。
- **密钥检查**：确认密钥是否带有多余空格，或者该密钥是否已经欠费。
- **模型匹配**：请确保你选择的模型名称与服务商提供的完全一致。你可以点击提供商列表右侧的「同步」按钮来自动拉取。

### 2. 悬浮球不显示或点击没反应

**Q: 启动服务后没有看到悬浮球？**
- **权限检查**：请确保已授予「显示在其他应用上层」权限。部分系统（如 MIUI/HyperOS）还需要在设置中手动开启「后台弹出界面」。
- **电池策略**：如果悬浮球运行一段时间后消失，可能是被系统后台清理了。请将 SolveX 加入系统的「电池优化白名单」。

### 3. 如何选择合适的截屏模式？

- **追求稳定**：推荐使用 **Shizuku (ADB)** 模式，它具有最高的成功率。
- **追求便捷**：使用 **无障碍截图** 模式，设置一次后即可永久免授权。
- **兼容性第一**：使用 **系统录屏** 模式，这在几乎所有 Android 设备上都能正常工作。

### 4. 识别效果不理想怎么办？

- **文字模糊**：如果是手写体或排版复杂的文档，请长按悬浮球切换到 **视觉多模态引擎**。
- **AI 幻觉**：如果助手给出的答案不正确，可以尝试在「设置 -> 助手管理」中修改该助手的提示词，要求其“分步骤详细推导”。

> 如果以上方法都无法解决你的问题，可以尝试进入「设置 -> 数据管理」执行「重置所有配置」，然后重新按照首页引导进行设置。
